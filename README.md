# Hello Volcanoes \- Client\-Side Website Development Project
The purpose of this project is to document the development process of the "Hello Volcanoes" website, created as part of the coursework for the Web Computing (CAB230) unit at Queensland University of Technology (QUT). This project is scored **88/100**

## Project Objectives
This project aimed to create an interactive and informative platform that provides users with comprehensive details about volcanoes worldwide, including their characteristics, eruption histories, and geographical locations. The website features user authentication, allowing registered users to access exclusive content and functionalities.

This project demonstrates the application of modern web development technologies, including **React, Ag-Grid-React, React-Chartjs-2, and Pigeon-Maps**, to deliver a responsive and engaging user experience. The development process involved careful planning, implementation, and testing to ensure the website meets its objectives while addressing potential limitations.

For a detailed overview of the project's development process, challenges, and reflections, please refer to the ***report.docx*** file included in the repository.

## How to Use
### 1. Clone the Repository
```
git clone https://github.com/anhnguyen765/HelloVolcanoes.git
```
### 2. Install Dependencies
```
npm install
```
### 3. Run the Development Server
```
npm run dev
```
### 4. Explore the Website
- Open your browser and navigate to the provided local server address (`https://localhost:5173`)
- Browse volcano information, register, or log in to access exclusive features
